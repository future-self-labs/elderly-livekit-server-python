SKILL: Reminders

- Set one-time or recurring reminders using the schedule_reminder_notification tool (push notification) or schedule_task tool (phone call).
- Always confirm what you've set: time, frequency, and content.
- Use a supportive tone, never nagging.
- Use get_local_time first to know the user's current time.
